# AWS Cloud Practitioner Exam Revision

A structured revision repository for the **AWS Certified Cloud Practitioner** exam.

This repo contains clear, module-by-module study notes and revision summaries based on the AWS learning modules. It is designed to help you revise faster, review important concepts, and refresh key exam topics before the test.

---

## What this repository includes

- Revision notes for each AWS Cloud Practitioner module
- Simplified explanations of core AWS concepts
- Exam-focused summaries
- Key services and comparisons
- Quick review material before the exam

---

## Modules Covered

### 1. Cloud Concepts Overview
Covers the fundamentals of cloud computing, cloud deployment models, and the value proposition of AWS.

### 2. Cloud Economics and Billing
Covers pricing fundamentals, total cost of ownership (TCO), AWS Organizations, billing tools, and support plans.

### 3. AWS Global Infrastructure Overview
Covers Regions, Availability Zones, edge locations, global infrastructure design, and AWS service categories.

### 4. AWS Cloud Security
Covers the shared responsibility model, IAM, account protection, encryption, compliance, and core AWS security services.

### 5. Networking and Content Delivery
Covers networking basics, Amazon VPC, routing, gateways, security groups, network ACLs, Route 53, and CloudFront.

### 6. Compute
Covers Amazon EC2, pricing models, cost optimization, containers, AWS Lambda, and Elastic Beanstalk.

### 7. Storage
Covers Amazon EBS, Amazon S3, Amazon EFS, and Amazon S3 Glacier, including use cases and comparisons.

### 8. Databases
Covers Amazon RDS, DynamoDB, Redshift, and Aurora, and explains when to use each database service.

### 9. Cloud Architecture
Covers the AWS Well-Architected Framework, reliability, availability, and AWS Trusted Advisor.

### 10. Auto Scaling and Monitoring
Covers Elastic Load Balancing, Amazon CloudWatch, Amazon EC2 Auto Scaling, and monitoring-driven scaling strategies.
